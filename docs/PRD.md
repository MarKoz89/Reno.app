# Reno App PRD

## Product Summary

Reno App is a consumer-facing renovation planning app that helps homeowners turn a room photo and a few guided answers into a practical renovation plan, style direction, and explainable cost estimate.

The product should feel like a guided planning assistant, not a contractor CRM, marketplace, or internal dashboard.

## Target User

- Homeowners or renters planning a room refresh or small renovation.
- Users who need early clarity before contacting contractors or buying materials.
- Users who may not know renovation terminology, cost drivers, or project sequencing.

## Core User Journey

1. Land on the homepage and understand the value quickly.
2. Upload one or more room photos.
3. Select a desired renovation style.
4. Answer a guided wizard about room type, size, scope, budget, timeline, and constraints.
5. Review a generated renovation plan with deterministic cost ranges.
6. Save the project and return to it later.

## MVP Promise

The MVP helps a user create a saved renovation planning session with:

- Uploaded room context.
- Selected style direction.
- Wizard answers.
- Estimated cost range.
- Recommended renovation priorities.
- Simple project summary.

## Non-Goals For MVP

- Contractor management.
- Client records.
- Internal dashboards.
- Payments.
- Scheduling contractors.
- Full construction documents.
- Marketplace purchasing.
- Multi-user collaboration.
- Database-backed accounts unless explicitly added later.

## Success Criteria

- A user can complete the full journey without assistance.
- Estimates are deterministic, explainable, and reproducible from the same inputs.
- AI output improves clarity and presentation but does not determine final pricing.
- Saved projects can be reopened in a later session.
